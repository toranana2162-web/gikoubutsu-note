# 技工物管理ノート

歯科医院向けの技工物（依頼〜納品）管理アプリ。紙のノートに近い見た目で、スマホ・PCどちらでも使えます。

## 特徴
- **サーバー不要・無料**：HTML/CSS/バニラJS のみ。データはブラウザの `localStorage` に保存。
- **ログイン不要**：ブラウザで `index.html` を開くだけ。
- **紙のノート風UI**：現場スタッフが受け入れやすいデザイン。

## 使い方
1. `index.html` をブラウザで開く（ダブルクリック、またはホスティングして共有）。
2. **登録**タブで技工物を登録。
3. **トップ**タブで「今日の納品予定」「期限超過」を一目で確認。
4. **一覧**タブで検索・絞り込み、納品済み／指示書チェック。

> 無料で共有したい場合は GitHub Pages / Netlify / Cloudflare Pages などに
> この3ファイルをそのまま配置すれば動きます。

## 実装済み機能（MVP）
| ID | 機能 |
|----|------|
| F-01 | 技工物登録（印象日・患者名・製作物内容・対象歯（歯式）・送るもの・納品予定日） |
| F-02 | 納品済みチェック／納品前 指示書チェック |
| F-03 | 技工物一覧表示 |
| F-04 | 納品ステータス管理（未納品⇔納品済み） |
| F-05 | 納品予定日検索 |
| F-06 | 患者名検索 |
| F-08 | 期限超過の視覚表示（赤色・本日納品は緑色） |

> ※ 技工所は1か所のため、技工所の登録・検索（F-07）は不要として除外。
> ※ 対象歯は Zsigmondy-Palmer（ジグモンディ・パーマー）方式の歯式チャートをタップして選択。
> 　「永久歯（1〜8）」「乳歯（A〜E）」をボタンで切替でき、混合歯列（両方選択）にも対応。
| F-09 | 編集（前倒しで実装） |
| F-10 | 削除（前倒しで実装） |
| F-11 | スマホ・PC レスポンシブ対応 |

## ファイル構成
- `index.html` … 画面構造（トップ／一覧／登録タブ）
- `styles.css` … 紙ノート風スタイル・レスポンシブ
- `app.js`     … データ処理（localStorage CRUD・検索・状態判定）

## データについて
- データは**開いているブラウザ内**に保存されます（端末間では共有されません）。
- 複数端末での共有が必要になった場合は、Phase2 でクラウド保存（例: Supabase 無料枠）への差し替えを想定。

## テスト
jsdom で登録〜検索〜ステータス更新〜編集〜削除の一連フローを自動検証済み（20項目）。
